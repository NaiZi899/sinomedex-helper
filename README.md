# Sinomedex 自动答题

Sinomedex 在线考试平台的答题辅助油猴脚本。自动调用 AI 接口完成题目，并支持自动提交和自动继续下一题。

## 功能

- **AI 自动答题** — 自动读取题目和选项，调用 DeepSeek API 分析并选择正确答案
- **自动提交** — 选完答案后自动点击"提交"按钮
- **自动继续** — 提交后自动点击"继续"/"下一题"进入下一道
- **可视面板** — 侧边栏显示答题记录（Q&A 日志），实时查看 AI 回答结果
- **API Key 显隐** — 密钥输入框支持明文/密文切换

## 安装（提示，本插件基于谷歌浏览器开发，其他浏览器会有些许问题）

1. 浏览器安装 [Tampermonkey](https://www.tampermonkey.net/) 扩展
2. 下载 **[Sinomedex-Helper.txt](https://raw.githubusercontent.com/NaiZi899/sinomedex-helper/main/Sinomedex-Helper.txt)**，下载完成后**拖入 Tampermonkey 扩展**中
3. 打开任意 Sinomedex 考试页面，点击右下角 **AI** 按钮打开面板
4. 填入你的 API Key 并保存

### 获取 API Key

本脚本默认使用 DeepSeek API，你需要：

1. 前往 [DeepSeek 开放平台](https://platform.deepseek.com/) 注册并获取 API Key
2. 在脚本面板中填入该 Key
3. 也可在配置中自定义其他兼容 OpenAI 格式的 API 地址

## 使用方法

1. 进入考试页面后，点击右下角 **AI** 按钮打开配置面板
2. 确保已填入有效的 API Key
3. 根据需要开关三个选项：
   - **自动选择答案** — AI 分析题目并自动点击正确选项
   - **选完后自动点击提交** — 选择后自动点击"提交"
   - **提交后自动点击继续** — 提交后自动点击"继续"进入下一题
4. 点击 **保存**，脚本会在检测到新题目时自动运行
5. 答题记录会实时显示在面板日志区

## 技术原理

- **沙箱绕过**：油猴脚本运行在隔离沙箱中，无法直接触发页面内 JS 事件。脚本通过注入 `<script>` 到页面 DOM，在页面上下文中执行点击操作，绕过沙箱限制。
- **事件模拟**：生成完整的 MouseEvent 序列（pointerdown → mousedown → pointerup → mouseup → click），模拟真实用户点击行为。
- **AI 集成**：通过 GM_xmlhttpRequest 调用 DeepSeek API（兼容 OpenAI 格式），使用医学考试专项 prompt 提高准确率。
- **配置持久化**：使用 GM_setValue/GM_getValue 存储配置，关闭页面后不丢失。
